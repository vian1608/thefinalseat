# Backend API Server

Express.js backend for Urgent Travel flight booking application.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Copy `.env.example` to `.env` and update with your credentials:
```bash
cp .env.example .env
```

3. Start the server:
```bash
npm start
# or for development:
npm run dev
```

## Environment Variables

See `.env.example` for all required environment variables.
